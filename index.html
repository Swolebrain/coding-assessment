Question 1

The first and most basic concept you will learn in programming is that of variables. A variable is just a name that can hold a value. Just like in math you had statements like y = 5 and so on, in programming you can create variables with values. If you wanted to create a variable named y with the value 5, you could do so by writing the following statement:



var y = 5;

You need the "var" keyword before the name of the variable in order to tell the programming language that you are creating a new variable.

In programming, you can do arithmetic. It is built in. The arithmetic operators used in programming are:

  • * which is used for multiplication
  • / which is used for division
  • + and â€“ which are used for addition and subtraction

Thus, if you have a statement like 5*5 in programming, this statement will evaluate to the number 25 (because * stands for multiplication and 25 is the result of multiplying 5 times 5).

Let's see an example:



var a = 5;
var b = 10;
var c = a + b;

After the code above runs, the resulting value of the variable named c is 15. This is so because c is equal to a + b but a is 5 and b is 10.

Variables don't always need to have numeric values. There is a data type in programming known as the "string" type, and it is used to store sequences of characters. Consider the following statement:

var a = "hello world";

In this statement, a variable named a is being created and its value is the sequence of characters (letters, numbers, and symbols) "hello world". Note, in this context even the empty space between the words "hello" and "world" counts as a character.

You can add strings together by using the plus sign, just like adding numbers. Adding strings will basically join them together. Consider the following example:

var a = "hello world";
var b = "she would like to call a lifeline";
var c = a+b;

Then the value of c is the string "hello worldshe would like to call a lifeline". Notice that the plus sign simply joins the strings together, it doesn't add any extra space in between them. With that in mind, consider the following code:

Question 4

A big part of programming is getting the computer to make decisions. For this, we have if statements. An if statement will have the following structure:

if (condition){
    ...some code here
}

Condition is an expression like for example x < 10. The value of condition is either true or false. Whenever it is true, then the code inside the curly braces will run. If condition is not true, then the code inside the braces will not run. Examples:

var x = 10;
if (x < 12){
    var x = 50;
}

After this code runs, the value of var x is 50.

var x = 10;
if (x < 5){
    var x = 50;
}

After this code runs, the value of var x does not get changed because the condition wasn't true (x was 10 and 10 is not less than 5).
